How It Works

Two slots, each pickable to any provider. Send a prompt → both slots run in parallel, responses stream side-by-side. Per-slot JSON overlay lets you pass any API parameter (temperature, tools, reasoning_effort, etc.) — merged into every request body for the session.

Configure at least one provider. Ollama works without a key if you have it running locally.

Keys encrypted locally via electron-store. Never transmitted except to each provider's own API.
